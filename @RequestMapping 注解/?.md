后台测试代码

```java
//ant风格路径
@RequestMapping("/a?a/testAnt")
public String testAnt() {
    return "success";
}
```

测试结果

可以发现，`/a?a/testAnt`能够匹配的路径有

- `/a1a/testAnt`
- `/aaa/testAnt`

不能匹配的路径有

- `/testAnt`
- `/aaaa/testAnt`
- `/a/a/testAnt`
- `/a?a/testAnt`

> 即证明，`?`修饰的路径，有且必须有一个字符代替`?`的位置，即只能匹配单个字符，且不能为`/`和`?`这两种特殊字符（因为`/`和`?`在 url 路径中比较特殊，除此之外其他单个字符均可），符合预期
