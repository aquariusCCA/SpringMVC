| 功能        | URL路径         | 请求方式     |
| :-------- | :------------ | :------- |
| 访问首页      | `/`           | `GET`    |
| 查询所有员工    | `/employee`   | `GET`    |
| 删除员工      | `/employee/1` | `DELETE` |
| 跳转到添加员工页面 | `/toAdd`      | `GET`    |
| 添加员工      | `/employee`   | `POST`   |
| 跳转到修改员工页面 | `/employee/2` | `GET`    |
| 修改员工      | `/employee`   | `PUT`    |
